## Description
This repository contains the data and code used to produce the analyses and figures for:
H. William Fennie, Steven M. Porter, Kelia E. Axler, Brooke Snyder, and Alison L. Deary. 2026. Increased temperature decreases starvation resiliency in first feeding sablefish (Anoplopoma fimbria). Canadian Journal of Fisheries and Aquatic Sciences.

This study examines how warming affects larval sablefish starvation resiliency by comparing the effects of a phenological match and mismatch between first-feeding larvae and zooplankton prey availability under current and anomalously warm conditions. Data were collected by the National Oceanic and Atmospheric Administration, Alaska Fisheries Science Center, EcoFOCI Program.
